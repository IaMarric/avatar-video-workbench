# Community

Avatar Video Workbench is most useful when people compare it with their own
image-to-video workflows and report what is missing.

## Where To Contribute

- Use [Issues](https://github.com/IaMarric/avatar-video-workbench/issues) for
  reproducible bugs, focused feature requests, docs gaps, and scoped tasks.
- Use [Discussions](https://github.com/IaMarric/avatar-video-workbench/discussions)
  for workflow feedback, integration ideas, and comparisons with ComfyUI or
  other backends.
- Start with labels such as `good first issue` and `help wanted` when looking
  for approachable tasks.

## Useful Feedback

Good feedback includes:

- the backend or workflow you are comparing against;
- which metadata fields are missing from reports;
- what failed during setup or validation;
- whether a proposed example is safe to publish without private media or model
  assets.

Do not share private source photos, generated identity media, model weights,
bucket names, project IDs, service account files, or tokens in issues,
discussions, or pull requests.
